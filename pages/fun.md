## /rank
### HTTP Request
`GET https://weeb-api.up.railway.app/rank?pfp=https://telegra.ph/file/e991bb4b535a0f1425aa0.jpg&tag=7788&username=Texas&level=9&xp=1500&experience=1300&rank=New%20Devil%20🎃`

### Query Parameters

| Parameter | Required |   type   | Description | 
|    ---    |    ---   |   ---    |     ---     |
| pfp       | true     | `string` | set Avatar |
| experience | true    | `number` | set CurrentXP |
| xp        | true     | `number` | set RequiredXP |
| rank      | true     | `string` | set Rank |
| level     | true     | `number` | set Level |
| tag       | true     | `number` | set Discriminator |
| username  | true     | `string` | set Username |
| progress  | false    | `string` | set ProgressBar |
| bg        | false    | `string` | set Background |
